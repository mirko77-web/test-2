la dimensione dei REM è 7 volte la dimensione del font di html (la dimensione in rem è calcolata rispetto alla dimensione del font dell'elemento html 1 rem = a 16px)

la dimensione dei EM è 7 volte il font size del elemnto genitore (la dimensione è calcolata rispetto al font size dell'elemento genitore -- 16px x7em )
